How to build

To build Pepris for Windows (7 or later), use the latest version of Visual Studio and the provided Pepris.sln solution file. Simply open the solution file on Visual Studio and press F5 to compile it.

Please note that the game may not run correctly due to the missing settings.xml file. You can simply fix it by opening the ingame settings menu and adjusting the board size and opacity.
